# git常用命令

## 克隆远程分支
	根据用户配置，这里url做了修改，@后边少了.com   
	git@github.com:changyongyong/doc.git修改为git@github:changyongyong/doc.git   
	``` bash
	git clone git@github:changyongyong/doc.git
	```
## 查询工程修改状态
	
	``` bash
	git status
	```
	该命令提示下一步操作，以及一些内容修改，或者出现的问题

##	增加文件
	``` bash
	git add -A
	```
	
## 提交修改
	``` bash
	git commit -am 'log'
	```
	
## 查看本地分支
	``` bash
	git branch
	```
